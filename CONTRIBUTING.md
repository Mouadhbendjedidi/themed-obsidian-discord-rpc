#### Getting Started

1. Fork the repository
2. Clone your fork locally:

   ```bash
   git clone https://github.com/<your-username>/themed-obsidian-discord-rpc.git
   ```
3. Create a new branch:

   ```bash
   git checkout -b feat/your-feature-name
   ```

#### Making Changes

* Keep your changes focused and minimal
* Follow the existing code structure and style

#### Commit Guidelines

* check [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) for more!
* Example:

  ```bash
  git commit -m "feat: adding a brilliant feature"
  ```

#### Submitting a Pull Request

1. Push your branch:

   ```bash
   git push origin feature/your-feature-name
   ```
2. Open a Pull Request targeting the `main` branch
3. Describe **what** you changed and **why**
4. Link related issues if applicable

## Testing & Building

### Install dependencies
1. you need to make sure you installed the required dependencies in order to test or build the plugin by running this command:
```bash
npm install
```
2. in order to live test the plugin you just need to run:

```bash
npm run dev
```
3. if you want to build the plugin just run:
```bash
npm run build
```
